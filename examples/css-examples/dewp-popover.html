Popover 핵심 사용 가이드

개요

트리거 주변에 떠서 도우미 콘텐츠를 표시합니다. 바깥 클릭/ESC로 닫힙니다.

JavaScript 생성

DEWP.createPopover({
  trigger: '#helpBtn',
  content: document.getElementById('helpContent'),
  placement: 'bottom' // top | bottom | left | right
});

API

DEWP.openPopover(triggerEl);
DEWP.closePopover(triggerEl);
DEWP.togglePopover(triggerEl);
DEWP.toggleByTrigger('#helpBtn');

접근성

  • ESC 닫기, 바깥 클릭 닫기
  • 트리거로 포커스 복귀

자세한 내용은 docs/components/popover.md 참고.
