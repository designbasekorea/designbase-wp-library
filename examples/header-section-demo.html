2. 섹션 컴포넌트 (Section)

워드프레스 어드민 페이지에서 콘텐츠를 구분하는 섹션 컴포넌트입니다.

기본 섹션

일반 설정

플러그인의 기본 동작을 설정합니다.

여기에 폼 요소들이 들어갑니다.

아이콘이 있는 섹션

🔒 보안 설정

보안 관련 옵션을 설정합니다.

보안 설정 내용이 여기에 들어갑니다.
