구조와 변형

1. HTML 구조 - 필수 요소

Card를 사용하기 위한 최소한의 HTML 구조입니다:

<!-- 기본 구조 -->
<div class="dewp-card">
  <div class="dewp-card-header">카드 제목</div>
  <div class="dewp-card-body">
    <p>카드 본문 내용입니다.</p>
  </div>
  <div class="dewp-card-footer">카드 푸터</div>
</div>

필수 요소:

  • dewp-card: 최상위 컨테이너 클래스 (필수)
  • dewp-card-header: 헤더 영역 (선택사항, 제목이나 이미지 등)
  • dewp-card-body: 본문 영역 (필수, 주요 콘텐츠)
  • dewp-card-footer: 푸터 영역 (선택사항, 액션 버튼 등)

주의사항: Card는 순수 CSS 컴포넌트입니다. JavaScript가 필요하지 않습니다.

2. 기본 사용법

최소 구조 (본문만)
<div class="dewp-card">
  <div class="dewp-card-body">
    <p>간단한 카드입니다.</p>
  </div>
</div>
완전한 구조 (헤더 + 본문 + 푸터)
<div class="dewp-card">
  <div class="dewp-card-header">
    <h3>카드 제목</h3>
  </div>
  <div class="dewp-card-body">
    <p>카드 본문 내용입니다.</p>
  </div>
  <div class="dewp-card-footer">
    <button class="dewp-button dewp-button--primary">액션</button>
  </div>
</div>

3. 카드 변형 (Variants)

카드에 다양한 스타일 클래스를 추가하여 디자인을 변경할 수 있습니다:

<!-- 기본 카드 (약한 그림자) -->
<div class="dewp-card">
  <div class="dewp-card-body">기본 카드</div>
</div>

<!-- Outlined 카드 (테두리만) -->
<div class="dewp-card dewp-card--outlined">
  <div class="dewp-card-body">Outlined 카드</div>
</div>

<!-- Elevated 카드 (강한 그림자) -->
<div class="dewp-card dewp-card--elevated">
  <div class="dewp-card-body">Elevated 카드</div>
</div>

<!-- Hover 카드 (마우스 오버 시 그림자 증가) -->
<div class="dewp-card dewp-card--hover">
  <div class="dewp-card-body">Hover 카드</div>
</div>

변형 설명:

  • dewp-card--outlined: 그림자 없이 테두리만 있는 스타일
  • dewp-card--elevated: 그림자가 강조된 스타일 (기본보다 더 강한 그림자)
  • dewp-card--hover: 마우스를 올리면 그림자가 증가하는 인터랙티브 스타일

4. 실제 사용 예시

예시 1: 제품 카드
<div class="dewp-card dewp-card--elevated">
  <div class="dewp-card-header">
    <img src="product.jpg" alt="제품 이미지" style="width:100%; height:200px; object-fit:cover;">
  </div>
  <div class="dewp-card-body">
    <h3>제품명</h3>
    <p>제품 설명입니다.</p>
    <div style="font-size:1.5em; font-weight:bold; color:var(--db-primary);">₩29,000</div>
  </div>
  <div class="dewp-card-footer">
    <button class="dewp-button dewp-button--primary">장바구니 담기</button>
  </div>
</div>
예시 2: 프로필 카드
<div class="dewp-card">
  <div class="dewp-card-body" style="text-align:center;">
    <img src="avatar.jpg" alt="프로필" style="width:80px; height:80px; border-radius:50%; margin-bottom:var(--db-gap-m);">
    <h3>사용자 이름</h3>
    <p style="color:var(--db-text-secondary);">사용자 설명</p>
    <div style="margin-top:var(--db-gap-m);">
      <button class="dewp-button dewp-button--primary dewp-button--s">팔로우</button>
    </div>
  </div>
</div>
예시 3: 통계 카드
<div class="dewp-card dewp-card--outlined">
  <div class="dewp-card-body">
    <div style="font-size:2em; font-weight:bold; color:var(--db-primary);">1,234</div>
    <div style="color:var(--db-text-secondary); margin-top:var(--db-gap-xs);">총 방문자</div>
  </div>
</div>
예시 4: 클릭 가능한 카드
<a href="/detail" class="dewp-card dewp-card--hover" style="text-decoration:none; color:inherit; display:block;">
  <div class="dewp-card-header">
    <img src="thumbnail.jpg" alt="썸네일" style="width:100%; height:150px; object-fit:cover;">
  </div>
  <div class="dewp-card-body">
    <h3>카드 제목</h3>
    <p>카드 설명입니다.</p>
  </div>
</a>

5. 카드 그리드 레이아웃

여러 카드를 그리드로 배치하는 방법:

<!-- CSS Grid 사용 -->
<div style="display:grid; grid-template-columns: repeat(auto-fill, minmax(240px, 1fr)); gap: var(--db-gap-m);">
  <div class="dewp-card">...</div>
  <div class="dewp-card">...</div>
  <div class="dewp-card">...</div>
</div>

<!-- Flexbox 사용 -->
<div style="display:flex; gap: var(--db-gap-m); flex-wrap:wrap;">
  <div class="dewp-card" style="flex:1; min-width:240px;">...</div>
  <div class="dewp-card" style="flex:1; min-width:240px;">...</div>
  <div class="dewp-card" style="flex:1; min-width:240px;">...</div>
</div>

6. 이미지와 함께 사용

<div class="dewp-card">
  <div class="dewp-card-header" style="padding:0;">
    <img src="image.jpg" alt="이미지" style="width:100%; height:auto; display:block;">
  </div>
  <div class="dewp-card-body">
    <h3>이미지가 있는 카드</h3>
    <p>이미지 위에 오버레이 텍스트를 넣을 수도 있습니다.</p>
  </div>
</div>

<!-- 이미지 오버레이 -->
<div class="dewp-card" style="position:relative;">
  <div class="dewp-card-header" style="padding:0; position:relative;">
    <img src="image.jpg" alt="이미지" style="width:100%; height:200px; object-fit:cover;">
    <div style="position:absolute; bottom:0; left:0; right:0; background:linear-gradient(transparent, rgba(0,0,0,0.7)); color:white; padding:var(--db-padding-m);">
      <h3 style="margin:0;">오버레이 제목</h3>
    </div>
  </div>
  <div class="dewp-card-body">
    <p>본문 내용</p>
  </div>
</div>

7. 접근성 (A11y)

Card는 웹 접근성을 고려하여 설계되었습니다:

  • 시맨틱 HTML: 적절한 HTML 요소를 사용하여 구조적 의미를 제공합니다.
  • 이미지 alt 텍스트: 카드 내 이미지는 항상 alt 속성을 포함해야 합니다.
  • 링크 카드: 카드 전체가 클릭 가능한 링크인 경우, 명확한 링크 텍스트를 제공하세요.
  • 키보드 접근: 링크나 버튼이 포함된 카드는 키보드로 접근 가능해야 합니다.
  • 색상 대비: 텍스트와 배경의 대비율이 WCAG AA 기준을 만족해야 합니다.

권장 사항:

  • 카드 제목은 <h2>~<h6> 태그를 사용하세요.
  • 카드가 클릭 가능한 경우, 포커스 스타일을 명확히 표시하세요.
  • 여러 카드가 있을 때는 각 카드에 고유한 식별자를 제공하세요.

8. 주의사항 및 팁

  • 본문 필수: dewp-card-body는 필수 요소입니다. 헤더와 푸터는 선택사항입니다.
  • dewp-card-header에 이미지를 넣을 때는 padding:0을 추가하여 여백을 제거할 수 있습니다.
  • 그리드 레이아웃: 여러 카드를 배치할 때는 CSS Grid나 Flexbox를 사용하세요.
  • 반응형: 작은 화면에서는 카드 너비를 조절하거나 세로로 쌓이도록 설정하세요.
  • 일관성: 같은 페이지 내의 카드는 동일한 변형을 사용하는 것이 좋습니다.
  • 콘텐츠 길이: 카드 본문이 너무 길면 잘라내거나 "더보기" 링크를 추가하세요.
  • 성능: 많은 카드를 렌더링할 때는 가상 스크롤링을 고려하세요.
