문서 요약

Modal 핵심 사용 가이드

개요

오버레이 위에 떠서 사용자 상호작용을 차단하고 메시지/폼을 표시합니다.

HTML 구조

<button class="dewp-btn dewp-btn-primary" id="openModal">모달 열기</button>

모달 자체는 JavaScript로 생성되므로, 트리거 버튼만 준비하면 됩니다. 여러 모달을 사용하려면 각 버튼에 고유한 ID를 부여하세요.

JS 사용

const trigger = document.getElementById('openModal');
trigger.addEventListener('click', () => {
  DEWP.showModal({
    title: '알림',
    content: '<p>모달 내용입니다.</p>',
    size: 'md',
    closeOnOverlayClick: true,
  });
});

// 제어 메서드
DEWP.closeModal(trigger);      // 특정 모달 닫기
DEWP.closeAllModals();         // 열린 모든 모달 닫기
DEWP.showConfirmModal({        // 헬퍼: 확인/취소 다이얼로그
  title: '삭제',
  content: '<p>정말 삭제할까요?</p>',
  onConfirm: () => console.log('confirmed'),
});

옵션

  • title, content, size(sm/md/lg)
  • closable, overlay, closeOnEscape, closeOnOverlayClick
  • 콜백: onOpen, onClose

긴 내용 처리

컨테이너는 flex 레이아웃으로 헤더/푸터 고정, 본문만 스크롤됩니다.

접근성

  • 모달 열릴 때 포커스 트랩, 닫힌 후 트리거로 포커스 복귀
  • 키보드: ESC 닫기, Tab 순환
  • ARIA: role="dialog", aria-modal="true", 라벨링
