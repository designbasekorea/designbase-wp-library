1. HTML 구조 - 기본 사용법

Button을 사용하기 위한 기본 HTML 구조입니다:

<!-- 기본 버튼 -->
<button class="dewp-button dewp-button--primary">Primary Button</button>

<!-- 링크를 버튼처럼 스타일링 -->
<a href="#" class="dewp-button dewp-button--primary">Link Button</a>

<!-- 비활성화 상태 -->
<button class="dewp-button dewp-button--primary" disabled>Disabled Button</button>

필수 요소:

  • dewp-button: 기본 버튼 클래스 (필수)
  • dewp-button--{variant}: 버튼 변형 클래스 (필수, primary, secondary, tertiary 등)

주의사항: Button은 순수 CSS 컴포넌트입니다. JavaScript가 필요하지 않습니다.

2. 버튼 변형 (Variants)

기본 변형
<button class="dewp-button dewp-button--primary">Primary</button>
<button class="dewp-button dewp-button--secondary">Secondary</button>
<button class="dewp-button dewp-button--tertiary">Tertiary</button>
상태별 변형
<button class="dewp-button dewp-button--success">Success</button>
<button class="dewp-button dewp-button--warning">Warning</button>
<button class="dewp-button dewp-button--danger">Danger</button>
<button class="dewp-button dewp-button--info">Info</button>
Outline 변형
<button class="dewp-button dewp-button--outline-primary">Outline Primary</button>
<button class="dewp-button dewp-button--outline-secondary">Outline Secondary</button>
<button class="dewp-button dewp-button--outline-success">Outline Success</button>
<button class="dewp-button dewp-button--outline-danger">Outline Danger</button>

3. 크기 변형

버튼 크기를 조절하는 클래스들:

<button class="dewp-button dewp-button--primary dewp-button--xs">Extra Small</button>
<button class="dewp-button dewp-button--primary dewp-button--s">Small</button>
<button class="dewp-button dewp-button--primary dewp-button--m">Medium (기본)</button>
<button class="dewp-button dewp-button--primary dewp-button--l">Large</button>
<button class="dewp-button dewp-button--primary dewp-button--xl">Extra Large</button>

크기 클래스:

  • dewp-button--xs: Extra Small (가장 작음)
  • dewp-button--s: Small
  • dewp-button--m: Medium (기본값, 클래스 생략 가능)
  • dewp-button--l: Large
  • dewp-button--xl: Extra Large (가장 큼)

4. 상태 변형

비활성화 상태
<button class="dewp-button dewp-button--primary" disabled>Disabled Button</button>
<button class="dewp-button dewp-button--outline-primary" disabled>Disabled Outline</button>

주의: disabled 속성은 <button> 요소에만 사용할 수 있습니다. 링크 요소에는 사용할 수 없으므로, 대신 클래스를 추가하여 스타일링하세요.

로딩 상태
<button class="dewp-button dewp-button--primary dewp-button--loading">
  <span class="dewp-button-text">Loading...</span>
</button>

로딩 상태 사용법:

  • dewp-button--loading 클래스를 추가하면 로딩 스피너가 표시됩니다.
  • 버튼 텍스트는 dewp-button-text 클래스를 가진 요소로 감싸야 합니다.
  • 로딩 중에는 버튼이 비활성화된 것처럼 보이지만, 실제로는 disabled 속성을 추가하는 것을 권장합니다.

5. 전체 너비 버튼 (Block)

<button class="dewp-button dewp-button--primary dewp-button--block">Block Button</button>

dewp-button--block 클래스를 추가하면 버튼이 부모 요소의 전체 너비를 차지합니다.

6. 아이콘과 함께 사용

<!-- 아이콘 폰트 사용 -->
<button class="dewp-button dewp-button--primary">
  <i class="icon-save"></i>
  <span>저장</span>
</button>

<!-- SVG 아이콘 사용 -->
<button class="dewp-button dewp-button--primary">
  <svg width="16" height="16" viewBox="0 0 24 24">
    <path d="M19 13h-6v6h-2v-6H5v-2h6V5h2v6h6v2z"/>
  </svg>
  <span>추가</span>
</button>

<!-- 아이콘만 있는 버튼 -->
<button class="dewp-button dewp-button--primary" aria-label="삭제">
  <i class="icon-delete"></i>
</button>

권장 사항:

  • 아이콘과 텍스트를 함께 사용할 때는 적절한 간격을 유지하세요.
  • 아이콘만 있는 버튼은 aria-label을 추가하여 접근성을 개선하세요.
  • 아이콘 크기는 버튼 크기에 맞게 조절하세요.

7. 버튼 그룹

<div class="dewp-btn-group" style="display:flex; gap: var(--db-gap-xs);">
  <button class="dewp-button dewp-button--primary">첫 번째</button>
  <button class="dewp-button dewp-button--primary">두 번째</button>
  <button class="dewp-button dewp-button--primary">세 번째</button>
</div>

버튼 그룹 스타일링:

  • dewp-btn-group 클래스를 사용하여 버튼들을 그룹화할 수 있습니다.
  • Flexbox를 사용하여 간격을 조절하세요: display:flex; gap: var(--db-gap-xs);
  • 필요에 따라 flex-wrap:wrap을 추가하여 반응형으로 만들 수 있습니다.

8. 실제 사용 예시

예시 1: 폼 제출 버튼
<form>
  <!-- 폼 필드들 -->
  <div class="dewp-btn-group" style="display:flex; gap: var(--db-gap-xs); margin-top: var(--db-gap-m);">
    <button type="submit" class="dewp-button dewp-button--primary">저장</button>
    <button type="button" class="dewp-button dewp-button--secondary">취소</button>
  </div>
</form>
예시 2: 액션 버튼
<div style="display:flex; gap: var(--db-gap-xs); flex-wrap:wrap;">
  <button class="dewp-button dewp-button--success">
    <i class="icon-check"></i>
    <span>승인</span>
  </button>
  <button class="dewp-button dewp-button--danger">
    <i class="icon-close"></i>
    <span>거부</span>
  </button>
  <button class="dewp-button dewp-button--tertiary">
    <i class="icon-info"></i>
    <span>상세보기</span>
  </button>
</div>
예시 3: 로딩 상태 버튼
<button id="submitBtn" class="dewp-button dewp-button--primary">
  <span class="dewp-button-text">제출하기</span>
</button>

<script>
document.getElementById('submitBtn').addEventListener('click', function() {
  const btn = this;
  btn.classList.add('dewp-button--loading');
  btn.disabled = true;
  
  // 비동기 작업 시뮬레이션
  setTimeout(() => {
    btn.classList.remove('dewp-button--loading');
    btn.disabled = false;
  }, 2000);
});
</script>

9. 접근성 (A11y)

Button은 웹 접근성을 고려하여 설계되었습니다:

  • 시맨틱 HTML: <button> 요소를 사용하여 버튼의 의미를 명확히 합니다.
  • 키보드 접근: 기본적으로 키보드 포커스 가능하며, Enter 또는 Space 키로 활성화됩니다.
  • 비활성화 상태: disabled 속성을 사용하면 키보드 포커스와 클릭이 모두 차단됩니다.
  • 아이콘 버튼: 아이콘만 있는 버튼은 aria-label을 추가하여 스크린 리더에 설명을 제공하세요.
  • 로딩 상태: 로딩 중에는 aria-busy="true"를 추가하는 것을 권장합니다.

권장 사항:

  • 버튼 텍스트는 명확하고 동작을 설명해야 합니다.
  • 링크를 버튼처럼 보이게 하려면 role="button"을 추가하세요.
  • 폼 제출 버튼은 type="submit"을 명시하세요.

10. 주의사항 및 팁

  • 버튼 vs 링크: 페이지 내 이동은 링크(<a>), 액션은 버튼(<button>)을 사용하세요.
  • 클래스 순서: 클래스 순서는 중요하지 않지만, 가독성을 위해 기본 클래스를 먼저 작성하는 것을 권장합니다.
  • 여러 변형 조합: 크기, 변형, 상태 클래스를 함께 사용할 수 있습니다: dewp-button dewp-button--primary dewp-button--l dewp-button--loading
  • 반응형: 작은 화면에서는 버튼 그룹에 flex-wrap:wrap을 추가하여 줄바꿈되도록 하세요.
  • 로딩 상태: 로딩 중에는 사용자가 버튼을 다시 클릭하지 못하도록 disabled 속성을 추가하세요.
  • 아이콘 크기: 버튼 크기에 맞게 아이콘 크기를 조절하세요. 일반적으로 버튼 텍스트 크기의 80-100%를 권장합니다.
  • 색상 대비: 접근성을 위해 버튼 텍스트와 배경색의 대비율이 WCAG AA 기준(4.5:1)을 만족해야 합니다.
