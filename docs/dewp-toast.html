문서 요약

Toast 핵심 사용 가이드

개요

짧은 알림 메시지를 화면 구석에 잠시 표시합니다.

API

DEWP.showSuccessToast('성공!', 2000);
DEWP.showWarningToast('경고!', 2000);
DEWP.showErrorToast('오류!', 2000);
DEWP.showInfoToast('정보', 2000);

옵션

  • 지속 시간(ms): 두 번째 인자(선택). 예: DEWP.showSuccessToast('완료', 2000)
  • 위치/컨테이너: 라이브러리가 자동으로 컨테이너를 생성/관리
