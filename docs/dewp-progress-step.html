문서 요약

Progress Step 핵심 사용 가이드

개요

여러 단계를 순차적으로 보여주는 진행 표시 UI입니다. 완료/현재/대기 상태를 구분합니다.

HTML 구조

<div class="dewp-progress-step">
  <div class="dewp-progress-step-item is-completed">
    <div class="dewp-progress-step-marker">1</div>
    <div class="dewp-progress-step-label">계정</div>
    <div class="dewp-progress-step-connector"></div>
  </div>
  <div class="dewp-progress-step-item is-active">
    <div class="dewp-progress-step-marker">2</div>
    <div class="dewp-progress-step-label">프로필</div>
    <div class="dewp-progress-step-connector"></div>
  </div>
  <div class="dewp-progress-step-item">
    <div class="dewp-progress-step-marker">3</div>
    <div class="dewp-progress-step-label">완료</div>
  </div>
</div>

상태 클래스

  • 완료: .is-completed
  • 현재: .is-active
  • 대기: 상태 클래스 미적용

크기 변형

  • .dewp-progress-step-sm, .dewp-progress-step-lg
