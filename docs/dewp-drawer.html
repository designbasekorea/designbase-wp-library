Drawer 핵심 사용 가이드

개요

화면 측면에서 슬라이드 인/아웃하는 패널. 오버레이/ESC로 닫기 옵션을 지원합니다.

생성/열기

const id = DEWP.createDrawer({ target: '#my-drawer', overlay: true, closeOnEscape: true, closeOnOverlayClick: true });
DEWP.openDrawer(id);

API

DEWP.closeDrawer(id);
DEWP.toggleDrawer(id);
DEWP.closeAllDrawers();

자세한 내용은 docs/components/drawer.md 참고.
