문서 요약

Radio Button 핵심 사용 가이드

개요

라디오 버튼은 여러 옵션 중 하나만 선택할 수 있는 폼 요소입니다.

Basic Radio

<div class="dewp-form-check dewp-form-radio">
  <input class="dewp-form-check-input" type="radio" name="group" id="radio1" value="option1">
  <label class="dewp-form-check-label" for="radio1">Option 1</label>
</div>

Radio Group

<div class="dewp-form-check-group">
  <div class="dewp-form-check dewp-form-radio">
    <input class="dewp-form-check-input" type="radio" name="size" id="small" value="small">
    <label class="dewp-form-check-label" for="small">Small</label>
  </div>
  <div class="dewp-form-check dewp-form-radio">
    <input class="dewp-form-check-input" type="radio" name="size" id="medium" value="medium" checked>
    <label class="dewp-form-check-label" for="medium">Medium</label>
  </div>
</div>

Inline Radio

<div class="dewp-form-check dewp-form-radio dewp-form-check-label-inline">
  <input class="dewp-form-check-input" type="radio" name="inline" id="yes" value="yes">
  <label class="dewp-form-check-label" for="yes">Yes</label>
</div>

Disabled Radio

<div class="dewp-form-check dewp-form-radio">
  <input class="dewp-form-check-input" type="radio" name="disabled" id="disabled" disabled>
  <label class="dewp-form-check-label" for="disabled">Disabled Option</label>
</div>

Error & Success States

<div class="dewp-form-group has-error">
  <div class="dewp-form-check dewp-form-radio">
    <input class="dewp-form-check-input" type="radio" name="error" id="error1">
    <label class="dewp-form-check-label" for="error1">Option 1</label>
  </div>
  <div class="dewp-form-error">Please select an option</div>
</div>

CSS 클래스

  • .dewp-form-check: 라디오 버튼 컨테이너
  • .dewp-form-radio: 라디오 버튼 스타일
  • .dewp-form-check-group: 라디오 버튼 그룹
  • .dewp-form-check-label-inline: 인라인 레이아웃
  • .has-error: 에러 상태
  • .has-success: 성공 상태
