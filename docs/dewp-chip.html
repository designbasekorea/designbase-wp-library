Chip 핵심 사용 가이드

1. HTML 구조 - 기본 사용법

Chip을 사용하기 위한 기본 HTML 구조입니다:

<!-- 기본 칩 -->
<span class="dewp-chip">기본</span>

<!-- 색상 변형 -->
<span class="dewp-chip dewp-chip-primary">Primary</span>
<span class="dewp-chip dewp-chip-success">Success</span>
<span class="dewp-chip dewp-chip-danger">Danger</span>

필수 요소:

  • dewp-chip: 기본 칩 클래스 (필수)
  • dewp-chip-{variant}: 색상 변형 클래스 (선택사항)

주의사항: Chip은 순수 CSS 컴포넌트입니다. JavaScript가 필요하지 않습니다.

2. 색상 변형

Chip에 다양한 색상 클래스를 추가하여 상태를 표현할 수 있습니다:

<!-- 기본 (회색) -->
<span class="dewp-chip">기본</span>

<!-- Primary (주요 색상) -->
<span class="dewp-chip dewp-chip-primary">Primary</span>

<!-- Success (성공/완료) -->
<span class="dewp-chip dewp-chip-success">완료</span>

<!-- Danger (위험/오류) -->
<span class="dewp-chip dewp-chip-danger">오류</span>

<!-- Warning (경고) -->
<span class="dewp-chip dewp-chip-warning">경고</span>

<!-- Info (정보) -->
<span class="dewp-chip dewp-chip-info">정보</span>

사용 가능한 색상:

  • dewp-chip-primary: Primary 색상 (기본 액션)
  • dewp-chip-success: Success 색상 (성공, 완료 상태)
  • dewp-chip-danger: Danger 색상 (오류, 삭제 등)
  • dewp-chip-warning: Warning 색상 (경고, 주의)
  • dewp-chip-info: Info 색상 (정보, 알림)

3. 실제 사용 예시

예시 1: 상태 표시
<div style="display:flex; gap: var(--db-gap-xs); align-items:center;">
  <span>주문 상태:</span>
  <span class="dewp-chip dewp-chip-success">배송 완료</span>
</div>

<div style="display:flex; gap: var(--db-gap-xs); align-items:center; margin-top: var(--db-gap-s);">
  <span>결제 상태:</span>
  <span class="dewp-chip dewp-chip-primary">결제 완료</span>
</div>
예시 2: 태그/카테고리
<div style="display:flex; gap: var(--db-gap-xs); flex-wrap:wrap;">
  <span class="dewp-chip">JavaScript</span>
  <span class="dewp-chip">React</span>
  <span class="dewp-chip">TypeScript</span>
  <span class="dewp-chip">Node.js</span>
</div>
예시 3: 필터 칩
<div style="display:flex; gap: var(--db-gap-xs); flex-wrap:wrap;">
  <button class="dewp-chip dewp-chip-primary" type="button">전체</button>
  <button class="dewp-chip" type="button">카테고리 1</button>
  <button class="dewp-chip" type="button">카테고리 2</button>
  <button class="dewp-chip" type="button">카테고리 3</button>
</div>
예시 4: 아이콘과 함께 사용
<span class="dewp-chip dewp-chip-success">
  <i class="icon-check"></i>
  <span>승인됨</span>
</span>

<span class="dewp-chip dewp-chip-danger">
  <i class="icon-close"></i>
  <span>거부됨</span>
</span>
예시 5: 제거 가능한 칩
<span class="dewp-chip">
  <span>선택된 항목</span>
  <button type="button" aria-label="제거" style="background:none; border:none; cursor:pointer; margin-left:var(--db-gap-xs);">
    <i class="icon-close"></i>
  </button>
</span>

4. 버튼으로 사용

Chip을 클릭 가능한 버튼으로 사용할 수 있습니다:

<button class="dewp-chip dewp-chip-primary" type="button">
  클릭 가능한 칩
</button>

<button class="dewp-chip" type="button" onclick="handleClick()">
  액션 칩
</button>

주의사항: 버튼으로 사용할 때는 type="button"을 명시하여 폼 제출을 방지하세요.

5. 링크로 사용

<a href="/category/javascript" class="dewp-chip">
  JavaScript
</a>

<a href="/tag/react" class="dewp-chip dewp-chip-primary">
  React
</a>

권장사항: 링크로 사용할 때는 명확한 텍스트를 제공하고, 호버 상태를 고려하세요.

6. 스타일링 가이드

Chip은 기본 스타일이 제공되지만, 필요에 따라 커스터마이징할 수 있습니다:

/* 칩 간격 조정 */
.dewp-chip + .dewp-chip {
  margin-left: var(--db-gap-xs);
}

/* 커스텀 색상 */
.dewp-chip.custom-color {
  background-color: #your-color;
  color: #your-text-color;
}

/* 크기 조정 */
.dewp-chip.small {
  font-size: 0.875rem;
  padding: var(--db-padding-xs) var(--db-padding-s);
}

.dewp-chip.large {
  font-size: 1.125rem;
  padding: var(--db-padding-m) var(--db-padding-l);
}

7. 접근성 (A11y)

Chip은 웹 접근성을 고려하여 설계되었습니다:

  • 시맨틱 HTML: <span>, <button>, <a> 등 적절한 요소 사용
  • 키보드 접근: 버튼이나 링크로 사용할 때는 키보드 포커스 가능
  • 색상 의존: 색상만으로 정보를 전달하지 말고, 텍스트나 아이콘을 함께 사용하세요
  • 제거 버튼: 제거 가능한 칩의 제거 버튼에는 aria-label을 추가하세요

권장 사항:

  • 칩의 텍스트는 명확하고 간결해야 합니다.
  • 여러 칩을 나열할 때는 적절한 간격을 유지하세요.
  • 색상 대비율이 WCAG AA 기준을 만족해야 합니다.

8. 주의사항 및 팁

  • 텍스트 길이: 칩은 짧은 텍스트를 위한 컴포넌트입니다. 긴 텍스트는 적절하지 않습니다.
  • 색상 의미: 색상은 일관된 의미를 가져야 합니다 (예: success=성공, danger=위험).
  • 그룹화: 관련된 칩들은 함께 그룹화하여 배치하세요.
  • 반응형: 작은 화면에서는 칩들이 자동으로 줄바꿈되도록 flex-wrap:wrap을 사용하세요.
  • 클릭 가능성: 칩이 클릭 가능한지 명확히 표시하세요 (호버 효과, 포커스 스타일 등).
  • 상태 표시: 동적 상태를 표시할 때는 적절한 색상 변형을 사용하세요.
  • 아이콘 사용: 아이콘과 함께 사용하면 시각적 정보 전달이 더 명확해집니다.
