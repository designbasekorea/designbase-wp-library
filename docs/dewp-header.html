문서 요약

Header 핵심 사용 가이드

개요

페이지 타이틀, 설명, 액션 버튼을 담는 상단 헤더입니다.

HTML 구조

<div class="dewp-header">
  <div class="dewp-header-content">
    <h1 class="dewp-header-title">제목</h1>
    <p class="dewp-header-description">설명</p>
  </div>
  <div class="dewp-header-actions">
    <button class="dewp-btn dewp-btn-outline-secondary">취소</button>
    <button class="dewp-btn dewp-btn-primary">저장</button>
  </div>
</div>

베스트 프랙티스

  • 액션은 우측 정렬, 기본/보조 버튼의 시각적 위계 유지
  • 설명은 한 줄 내 가독성 유지
  • 상세 페이지에는 .dewp-header-with-back과 .dewp-header-back-button 조합으로 뒤로가기 버튼을 제공합니다.
