Progress 핵심 사용 가이드

개요

작업 진행 정도를 백분율로 시각화합니다.

HTML 구조

<div class="dewp-progress">
  <div class="dewp-progress-bar" style="width:50%"></div>
</div>

JS 제어

const bar = document.querySelector('.dewp-progress-bar');
bar.style.width = '75%';

접근성

  • 필요 시 role="progressbar"와 aria-valuenow/min/max 속성 부여
