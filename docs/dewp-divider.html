Divider 핵심 사용 가이드

1. HTML 구조 - 기본 사용법

Divider를 사용하기 위한 기본 HTML 구조입니다:

<!-- 제목이 있는 디바이더 -->
<div class="dewp-divider-with-title">
  <span class="dewp-divider-title">섹션 제목</span>
</div>

<!-- 단순 구분선 -->
<hr class="dewp-divider" />

필수 요소:

  • dewp-divider-with-title: 제목이 있는 디바이더 컨테이너 (선택사항)
  • dewp-divider-title: 디바이더 제목 (제목형 디바이더 사용 시 필수)
  • dewp-divider: 구분선 클래스 (단순 구분선 사용 시 필수)

주의사항: Divider는 순수 CSS 컴포넌트입니다. JavaScript가 필요하지 않습니다.

2. 제목형 디바이더

제목과 함께 섹션을 구분하는 디바이더:

<div class="dewp-divider-with-title">
  <span class="dewp-divider-title">일반 설정</span>
</div>
<p>설정 내용이 여기에 표시됩니다.</p>

<div class="dewp-divider-with-title">
  <span class="dewp-divider-title">고급 설정</span>
</div>
<p>고급 설정 내용이 여기에 표시됩니다.</p>

사용 시나리오:

  • 폼의 여러 섹션을 구분할 때
  • 설정 페이지의 카테고리를 나눌 때
  • 콘텐츠를 논리적으로 그룹화할 때

3. 단순 구분선

제목 없이 단순히 콘텐츠를 구분하는 선:

<p>첫 번째 섹션의 내용입니다.</p>
<hr class="dewp-divider" />
<p>두 번째 섹션의 내용입니다.</p>

사용 시나리오:

  • 리스트 항목 간 구분
  • 간단한 콘텐츠 구분
  • 제목이 필요 없는 경우

4. 실제 사용 예시

예시 1: 폼 섹션 구분
<form>
  <div class="dewp-divider-with-title">
    <span class="dewp-divider-title">기본 정보</span>
  </div>
  <div>
    <label>이름</label>
    <input type="text">
  </div>
  
  <div class="dewp-divider-with-title">
    <span class="dewp-divider-title">연락처 정보</span>
  </div>
  <div>
    <label>이메일</label>
    <input type="email">
  </div>
</form>
예시 2: 설정 페이지
<div class="dewp-divider-with-title">
  <span class="dewp-divider-title">계정 설정</span>
</div>
<!-- 계정 설정 내용 -->

<div class="dewp-divider-with-title">
  <span class="dewp-divider-title">알림 설정</span>
</div>
<!-- 알림 설정 내용 -->

<div class="dewp-divider-with-title">
  <span class="dewp-divider-title">개인정보 설정</span>
</div>
<!-- 개인정보 설정 내용 -->
예시 3: 리스트 구분
<ul>
  <li>항목 1</li>
  <li>항목 2</li>
</ul>
<hr class="dewp-divider" />
<ul>
  <li>항목 3</li>
  <li>항목 4</li>
</ul>
예시 4: 카드 내부 구분
<div class="dewp-card">
  <div class="dewp-card-body">
    <h3>카드 제목</h3>
    <p>첫 번째 내용</p>
    <hr class="dewp-divider" />
    <p>두 번째 내용</p>
  </div>
</div>

5. 스타일링 가이드

Divider는 기본 스타일이 제공되지만, 필요에 따라 커스터마이징할 수 있습니다:

/* 디바이더 간격 조정 */
.dewp-divider {
  margin: var(--db-gap-l) 0;
}

/* 제목 스타일 커스터마이징 */
.dewp-divider-title {
  font-size: 1.125rem;
  font-weight: 600;
  color: var(--db-text-primary);
}

/* 구분선 색상 변경 */
.dewp-divider {
  border-color: var(--db-border-light);
}

/* 제목형 디바이더 여백 조정 */
.dewp-divider-with-title {
  margin: var(--db-gap-xl) 0 var(--db-gap-m) 0;
}

6. 접근성 (A11y)

Divider는 웹 접근성을 고려하여 설계되었습니다:

  • 시맨틱 HTML: <hr> 요소를 사용하여 구조적 의미 제공
  • 제목 사용: 제목형 디바이더는 섹션의 의미를 명확히 전달합니다
  • 스크린 리더: <hr> 요소는 스크린 리더에서 구분선으로 인식됩니다
  • 시각적 구분: 색상과 스타일로 명확한 시각적 구분 제공

권장 사항:

  • 제목형 디바이더는 적절한 제목 텍스트를 제공하세요.
  • 너무 많은 디바이더는 콘텐츠를 과도하게 분할할 수 있으므로 적절히 사용하세요.
  • 디바이더 주변에 적절한 여백을 유지하세요.

7. 주의사항 및 팁

  • 과도한 사용: 디바이더를 너무 많이 사용하면 콘텐츠가 조각나 보일 수 있습니다.
  • 일관성: 같은 페이지 내에서는 동일한 스타일의 디바이더를 사용하세요.
  • 여백 관리: 디바이더 위아래에 적절한 여백을 두어 시각적 균형을 유지하세요.
  • 제목 텍스트: 제목형 디바이더의 텍스트는 명확하고 간결해야 합니다.
  • 반응형: 작은 화면에서는 디바이더의 여백을 조정할 수 있습니다.
  • 색상 대비: 디바이더 선의 색상이 배경과 충분한 대비를 가져야 합니다.
  • 대체 방법: 간단한 구분이면 여백만으로도 충분할 수 있습니다.
